# User Journey

## Onboarding
1. User sets goals, dietary preferences, and restrictions.
2. The app asks about routines, cravings, and weak moments.
3. Optional integrations are connected: wearables, calendar, health data.
4. The system builds an initial profile and habit baseline.

## Daily Flow
- Morning: summary of the day, personalized meal ideas, and habit reminders.
- Pre-meal: predictive risk assessment and a supportive nudge if needed.
- Meal time: healthier swap suggestions and quick logging.
- Evening: reflection on choices, mood, and progress insights.

## Habit Loop
1. Detect context and risk factors.
2. Predict unhealthy behavior before it happens.
3. Send a timely nudge or recommendation.
4. User responds, accepts, or customizes the suggestion.
5. System learns and refines future guidance.
